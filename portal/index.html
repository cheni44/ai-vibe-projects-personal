Mosquito Detector

Web AI / Vision Audio

Real-time mosquito detection in your browser. Point your camera, and a synthesised buzz alert gets louder as mosquitoes get closer. No install required — runs entirely on-device.

  • 📷 Live camera preview with bounding-box overlay
  • 🔊 Web Audio API proximity buzz (no audio file)
  • 🎚️ Adjustable sensitivity slider
  • 🔄 Front / rear camera toggle

Mosquito Detector iOS

iOS Python AI / Vision

The same mosquito detector as a Pythonista 3 Python app on iPhone/iPad. Uses AVFoundation via objc_util for camera access and PIL heuristics for detection.

  • 🍎 Runs inside Pythonista 3 (no App Store needed)
  • 📐 PIL + NumPy blob-detection pipeline
  • 🔊 Proximity-scaled sound.Player alert
  • 🧪 Unit-tested detection helpers
